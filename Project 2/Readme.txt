Main File :
	Filters.java (Project2/src)

Executing the program : 

1) In terminal go to the location where the program file : Filters.java
2) Enter the command javac Filters.java to compile the code.
3) Enter the command java Filters to run the code.
4) Enter the inputs for Number of elements, size of Bit Array, Number of hashes, Type of hashing(bloom_filter or counting_bloom_filter or coded_bloom_filter), Number of filters,
and Number of steps.
5) After all the inputs are given respective filter output files(bloom_filter.txt, coded_bloom_filter.txt, and counting_bloom_filter.txt) will be created for type of filter you entered
as input.

There are four files in src folder. 

1) Filters.java : It has the code that is implemented for Bloom Filter, Counting Bloom Filter, Coded Bloom filter.

2) bloom_filter.txt: This file contains the No of elements of A in the filter and No of elements of B in the filter.

3)counting_bloom_filter.txt: This file contains No of elements of A in the filter.

4)coded_bloom_filter.txt: No of elements of A in the filter with the correct lookup.
